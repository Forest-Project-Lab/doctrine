# 判断の層（技能と候補語抽出）が決定論を補い保証限界を明示する

## 要求文

機械では割り切れず守れない部分を、技能（Skill。人の判断を支援する成果物）と候補語抽出が補える。各技能は、何を予防し、何を検出し、何を人に委ねるかを書く。候補語抽出は候補を出すだけで、どれを採るかは人が決める。こうして、ガードとリンタが届かない判断の層でも、どこまで保証するかが明示される。`[R9]`

## 優先度

中

## 受入基準参照

§6 の R9 行（各成果物に、予防・検出・委ねるの区別が書かれていること）。TEST-016 で確認する。

## 出所

spec §4.1（Skill は人の判断を支援し、機械で割り切れる処理は scripts に任せる）・§7（保証限界）・§4.3（候補の採否は人が決める）。
