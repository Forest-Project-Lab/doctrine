# 禁止同義語セルの末尾注記の扱い

## 背景

辞書の禁止同義語セルには、末尾に `（…）` の注記が付くものがある。`インターフェース（単独語）` は素のトークンを弾きたい。一方 `差し替え（操作名としては可。状態名は置換）` は、使ってよい文脈があるため、いつでも弾くわけにはいかない[R6]。

## 却下した選択肢

- 末尾注記を一律に削って文字列照合する案。使ってよい文脈のある語まで弾いてしまい、擬陽性を生むため却下した。
- 注記の付くセルを一律に文脈依存（文字列照合の対象にしない）とする案。`インターフェース（単独語）` のような確実な禁止語まで見逃すため却下した。

## 決定

末尾注記に「可」を含むセルは、文脈依存とみなして文字列照合の対象にしない。「可」を含まない注記は、素のトークンを取り出して文字列として照合する。

## 帰結

`インターフェース` は弾き、`差し替え` は弾かない。使ってよい文脈かどうかの最終判断は doc-review に委ねる。
