# 見つけやすさ（投影を正本から決定論で描画）

## 要求文

全体図と索引は、モデルから描画する投影として得られねばならない `[R1]`。ここでモデルとは、§3 の登録簿と各文書のフロントマターを指す。投影は手で書き溜めない。同じ源から描き直せば、結果はバイト単位で一致する（冪等）。源が変わらなければ、投影の `updated` も動かない。こうして、投影が現行集合からずれること（投影ドリフト）を防ぐ。

## 優先度

高

## 受入基準参照

TEST-014（投影の決定論描画・冪等・`--check` のドリフト検出）。

## 出所

spec §3.9「全体図と索引はモデルから描画する投影とし、手で保守しない」。要求タグ `[R1]`。
