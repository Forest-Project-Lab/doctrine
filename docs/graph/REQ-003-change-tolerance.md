# 変更耐性（影響集合を依存から列挙する）

## 要求文

ある文書を変えるとき、変更が波及する文書の集合（前向き影響集合）を、impacts 端をたどった推移閉包として列挙できること。依存が循環していても止まらずに走査を打ち切り、現行文書からの逆依存の数を数えられること。[R4]

## 優先度

高

## 受入基準参照

TEST-006 で確認する（前向き影響集合、循環があっても止まる走査、現行文書からの逆依存）。

## 出所

spec §3.8（降格不変条件、すなわち現行文書からの逆依存がゼロであること）と、変更フローが求める影響列挙に由来する。
